GlobeTrotter – Personalized Real-Time Travel Planning Platform

GlobeTrotter is a real-time, full-stack travel planning web application designed to simplify and enhance the way users plan multi-city trips. The platform enables travellers to create, organize, visualize, and share personalized itineraries while staying informed about budgets, schedules, and activities—all in one place.
Built with a user-centric design and powered by a relational database, GlobeTrotter transforms complex travel planning into an intuitive and interactive experience.

Key Features
 1. User Authentication
Secure signup, login, and password recovery.
2. Dashboard
View upcoming trips, recent itineraries, budget highlights, and recommended destinations.
3. Multi-City Trip Planning
Create customized trips with multiple cities, travel dates, and descriptions.
4. Interactive Itinerary Builder (Real-Time)
Add cities, assign activities day-wise, reorder plans dynamically, and see updates instantly.
5. City & Activity Discovery
Search and explore cities and activities with filters based on cost, duration, and category.
6. Automatic Budget Estimation
Real-time cost calculation with detailed breakdowns and over-budget alerts.
7. Calendar & Timeline View
Visualize trips using structured timelines or calendar-based layouts.
8. Trip Sharing
Generate public, read-only links to share itineraries or copy trips for inspiration.
9. User Profile & Settings
Manage personal information, preferences, and privacy controls.
10. Admin Analytics (Optional)
Monitor platform usage, popular destinations, and user engagement.

Tech Stack

Frontend: React (Responsive UI)
Backend: Node.js + Express
Database: PostgreSQL (Relational)
Real-Time Updates: WebSockets / Firebase
Authentication: Email & Password-based Auth

Project Objective

The goal of GlobeTrotter is to build a scalable, real-world travel planning solution that combines real-time interactivity, structured data management, and visual clarity, making travel planning as exciting as the journey itself.

Use Cases

Students and solo travellers planning budget-friendly trips
Families organizing multi-city vacations
Travel enthusiasts sharing itineraries with others
Hackathon / academic full-stack project demonstration
